# Install instructions

## Dependencies

**Docker has to be installed and running.**

MAC: https://docs.docker.com/docker-for-mac/install/

Linux - Fedora: https://docs.docker.com/install/linux/docker-ce/fedora/


## Executing
just type

``docker-compose up``

## Language

NODE_VERSION latest


## Work Directory

/usr/src/app
